# Troubleshooting

## REPORT_SOURCE_MODEL_MISSING
Earlier Part is not applied/registered.

## ROLE_SCOPE_REQUIRED
Owner must assign Part 124 Branch scope.

## TEACHER_CLASS_SCOPE_MISMATCH
Teacher can report only assigned Classes.

## PARENT_CHILD_SCOPE_MISMATCH
Parent can report only linked children.

## REPORT_EXPORT_ACCESS_DENIED
Non-owner can export only their own report.

## EXPORT_TOO_LARGE
Use narrower filters.

## Download returns 410
Ticket was used, expired, revoked or invalid. Issue a new ticket.

## EXPORT_FORMAT_NOT_SUPPORTED
Use CSV, JSON or HTML—not PDF/XLSX.
