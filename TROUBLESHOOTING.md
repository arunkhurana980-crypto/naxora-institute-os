# Troubleshooting

- `PART120_IDENTITY_MODEL_MISSING`: Part 120 not registered.
- `PART124_SCOPE_MODEL_MISSING`: Part 124 not registered.
- `BRANCH_NOT_FOUND`: create/use valid Part 128 Branch ID.
- `COURSE_NOT_FOUND`: create Course first.
- `ACCOUNT_IDENTIFIER_EXISTS`: use unique login ID.
- `WEAK_TEMPORARY_PASSWORD`: 10+ characters, letter and number.
- `EXACT_CONFIRMATION_REQUIRED`: copy exact preview confirmation.
- `OWNER_VERIFICATION_FAILED`: private Owner Action Secret wrong.
