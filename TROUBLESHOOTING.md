# Part 129 Troubleshooting

## `BULK_IMPORT_MODELS_MISSING`
Apply and verify Parts 120, 124 and 128 before Part 129.

## `IMPORT_VALIDATION_ERRORS_PRESENT`
Open the Errors tab, correct the file and create a fresh preview.

## `DUPLICATE_FOUND`
Change duplicate values or select the explicit `skip` policy.

## `IMPORT_REFERENCE_DRIFT`
A linked record changed after preview. Create a fresh preview.

## `WEAK_TEMPORARY_PASSWORD`
Use at least 10 characters with one letter and one number.

## `OWNER_VERIFICATION_FAILED`
Enter the private Owner Action Secret. Never send it in chat or screenshots.

## `rollbackApplied: true`
Partially created records were removed after failure. Correct the file and create a fresh preview.
