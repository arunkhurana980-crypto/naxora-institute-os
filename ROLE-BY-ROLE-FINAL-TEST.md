# Role-by-Role Final Test

Use generated identifiers and the shared private demo password.

## Owner
- Open `/app`.
- Owner Workspace.
- Account Access.
- Final Demo & Freeze.
- VANI Actions.
- Integration Centre.

## Branch Manager
- Branch Workspace.
- Check only assigned branch.
- Create Branch Task through VANI.

## Teacher
- Teacher Workspace.
- Check linked Students/Classes.
- Mark attendance.
- Create assignment.
- Send role-safe message.

## Student
- Student Workspace.
- Check classes, attendance, fees, assignments and results.
- Submit a Part 126 assignment.
- Create attendance correction or fee-assistance request.

## Parent
- Parent Workspace.
- Check only linked child.
- Create correction/assistance request.
- Send permitted message.

## Accountant
- Accountant Workspace.
- Check fees/invoices/receipts.
- Send fee reminder.

## Counsellor
- Counsellor Workspace.
- Check leads/follow-ups.
- Schedule admission follow-up.

## Staff
- Staff Workspace.
- Check assigned branch data.
- Test permitted attendance/message action.

Every cross-role or cross-branch negative test must remain blocked.
