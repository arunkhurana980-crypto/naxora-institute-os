# Razorpay Live Dashboard Checklist

Complete this with the adult legal merchant account holder.

## Account
- Account activation completed.
- Legal business details verified.
- Bank/settlement details confirmed.
- Website/app submitted and approved.
- Required payment methods reviewed.

## Website
Publish working HTTPS pages:
- Pricing.
- Terms and Conditions.
- Privacy Policy.
- Contact Us.
- Cancellation and Refunds.
- Shipping/Delivery policy or a clear not-applicable digital-service policy.

## Live credentials
- Switch Razorpay Dashboard to Live Mode.
- Generate Live API keys.
- Save them once in private password storage/Render.
- Never commit or screenshot the Key Secret.

## Live webhook
- Create a separate Live webhook.
- Use a separate Live webhook secret.
- Configure alert email.
- Select required Subscription events.
- Validate signatures and delivery.

## Launch
- Save Test environment backup.
- Approve Part 118 launch preview.
- Perform supervised manual switch.
- Start with a controlled low-value transaction chosen by the adult merchant.
- Verify provider status, webhook and settlement records.
- Roll back if any verification fails.
