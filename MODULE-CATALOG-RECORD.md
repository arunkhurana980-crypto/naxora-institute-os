# Module Catalog Record — Part 119

The catalog contains:
- Role apps.
- Institute modules.
- Teaching modules.
- AI/VANI modules.
- Business modules.
- Parts 113–118 billing controls.
- NAXORA OS 3.0 placeholders protected by Part 116 V3 access.

Routes are server-controlled. Client-submitted arbitrary routes are not accepted.

Some legacy route names may require mapping fixes during Parts 121–124 when each role module is consolidated. A missing legacy route does not grant access; it displays the existing 404 response inside the shell and must be corrected in that role's consolidation part.
