# Part 136 Security Boundary

- Role tests use the current logged-in session.
- Role JWTs are not saved in acceptance records.
- Do not paste tokens into fixture JSON.
- Positive checks create preview only, then cancel.
- Outside-scope access must fail.
- Workflow evidence is read from Part 135 MongoDB records.
- Certificate is an internal product acceptance record.
- Certificate is not external security, legal, financial or accessibility certification.
- Payment settlement is not certified.
- Password/OTP/Owner-secret handling remains excluded.
