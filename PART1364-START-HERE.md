# Part 136.4 — Simple Owner Signup and Login

Owner Login:

```text
Owner Email/Phone/Login ID
Password
```

Owner ko Institute ID enter nahi karni.

New Institute:

```text
Institute name
Owner name
Owner Email/Phone/Login ID
Password
→ Institute ID automatic
→ Owner account automatic
→ Direct /app session
```

Other roles continue using `/common-login`.
