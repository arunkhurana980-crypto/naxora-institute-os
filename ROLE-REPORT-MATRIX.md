# Part 134 Role Matrix

- Owner: all institute reports
- Branch Manager: assigned Branch reports
- Teacher: assigned Class attendance/academic, assigned Student 360, own communication delivery
- Accountant: assigned Branch finance and own communication delivery
- Counsellor: assigned CRM Leads and own communication delivery
- Staff: assigned Branch and attendance
- Student: own attendance, academic, finance and Student 360
- Parent: Owner-linked child reports

Non-owner roles can export only reports they generated.
