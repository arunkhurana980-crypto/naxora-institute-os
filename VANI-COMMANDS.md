# Part 129 VANI Commands

```text
VANI, CSV import kholo
```

```text
VANI, Students ka bulk data upload karna hai
```

```text
VANI, linked JSON package import screen kholo
```

```text
VANI, Teachers ki spreadsheet import karni hai
```

Expected result:

```text
VANI opens VANI Bulk Import
→ Owner manually selects file
→ mapping
→ validation
→ preview
→ exact confirmation
→ private password/Owner Secret
→ chunked import
→ audit or rollback result
```

VANI does not secretly access a local file and does not accept passwords or secrets in commands.
