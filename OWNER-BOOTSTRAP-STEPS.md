# First Owner Bootstrap Steps

1. Open:

```text
https://naxora-institute-os.onrender.com/owner-bootstrap
```

2. Enter:
   - Institute name.
   - Owner full name.
   - Owner Email, Phone or Login ID.
   - Private bootstrap secret.

3. Click `Generate Institute ID Preview`.

4. Review the automatically generated Institute ID.

5. Enter/copy the exact confirmation.

6. Create a password:
   - Minimum 10 characters.
   - At least one letter.
   - At least one number.

7. Click `Create First Owner`.

8. Privately save the generated Institute ID.

9. Open `/app` or use `/common-login`.

Never send the password or bootstrap secret in chat.
