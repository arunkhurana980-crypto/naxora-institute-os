# Export Security

- CRM phone/email excluded
- encrypted communication address excluded
- finance payment references excluded
- password, OTP, JWT, provider secret and identity-document numbers excluded
- max 1000 report rows
- max 2 MB export
- export expires after 24 hours
- ticket expires after 5 minutes
- ticket is one-time
- export content is hidden from list/history APIs
- PDF/XLSX not supported in Part 134
