# Free-First Deployment Plan

## Phase 1: Local demo
Run backend locally and use `/app` hosted frontend.

## Phase 2: Free online demo
- MongoDB Atlas free tier
- Backend on free/low-cost available option
- Frontend on Vercel/Netlify free, or same backend `/app`
- No paid domain initially
- Razorpay test mode only

## Phase 3: First client
When first paid client is ready, consider stable hosting and domain.

## Security
Before any public demo, rotate exposed MongoDB/Razorpay credentials.
