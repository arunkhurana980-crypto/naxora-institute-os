# Part 130 Troubleshooting

## `PART128_MODELS_MISSING`
Apply and register Part 128 before Part 130.

## `TEACHER_CLASS_SCOPE_MISMATCH`
The logged-in Teacher is not assigned to the selected Part 128 Class.

## `STUDENT_CLASS_MISMATCH`
Use a Student linked to the selected Class.

## `MARKS_EXCEED_MAX_MARKS`
Correct marks so every score is at or below Exam maxMarks.

## `MARKS_REQUIRED_BEFORE_PUBLISH`
Record at least one Student mark before publishing result.

## `EXACT_CONFIRMATION_REQUIRED`
Copy the exact confirmation generated by preview.

## `DUPLICATE_ACADEMIC_ACTION`
The same action was executed inside the duplicate-protection window.

## `rollbackApplied: true`
A partial write was reverted after execution failure. Review the error and retry.
