# Client Onboarding Checklist

1. Institute name, owner name, phone, email collect karo.
2. Branch/city/address details lo.
3. Current students, courses, batches aur fees structure samjho.
4. Admin login create karo.
5. 10–20 demo students import/add karo.
6. 2–3 batches create karo.
7. Fees, attendance aur enquiries module demo karo.
8. Live Classes add-on chahiye ya nahi confirm karo.
9. Online Batch Access pricing set karo.
10. Payment/Razorpay setup test mode me verify karo.
11. Owner ko 30-minute training do.
12. First 7 days feedback lo.
