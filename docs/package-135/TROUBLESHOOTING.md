# Troubleshooting

- `WORKFLOW_INTENT_NOT_RECOGNISED`: use built-in workflow phrase or `actions=`.
- `CUSTOM_ACTION_NOT_AVAILABLE`: native catalog action unavailable.
- `WORKFLOW_FIELDS_REQUIRED`: answer requested field.
- `WORKFLOW_DEPENDENCY_PENDING`: complete previous step.
- `NATIVE_PREVIEW_REQUIRED`: create preview first.
- `EXACT_CONFIRMATION_REQUIRED`: copy exact text.
- Native failure: correct value/scope, send `retry`, create fresh preview.
- Catalog below 60: one or more Parts 130–134 are inactive.
- Do not enter passwords or Owner secrets into Part 135.
