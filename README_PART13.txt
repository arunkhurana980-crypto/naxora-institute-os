PART 13 - Certificates & ID Cards System

Backend APIs:
GET    /api/certificates
POST   /api/certificates
GET    /api/certificates/:id
PUT    /api/certificates/:id
PATCH  /api/certificates/:id/status
DELETE /api/certificates/:id

Frontend:
frontend/certificates.html
frontend/certificates.css
frontend/certificates.js

Run:
1. Copy your working backend/.env into this backend folder.
2. taskkill /F /IM node.exe
3. cd backend
4. npm install
5. npm run dev
6. Open frontend/index.html with Live Server.

Expected terminal line:
✅ PART 13 ROUTES ACTIVE: /api/certificates
