# Role Scope Setup Guide

## Open
`/role-scope-manager`

## Parent
Use the actual Student record IDs used by your current MongoDB models.

```text
Parent Account
→ Child Student IDs
→ Preview
→ Exact confirmation
→ Owner Action Secret
→ Save
```

## Branch Manager
Use Branch IDs stored on branch-scoped operational records.

## Accountant, Counsellor and Staff
Choose one:
- specific Branch IDs, or
- explicit institute-wide access.

Institute-wide access is never inferred automatically.

## Changing or disabling
Set Status to Disabled and save with confirmation. The workspace will return Scope Pending until an active assignment exists.

## Privacy
Do not place Owner Action Secret, passwords or OTPs in screenshots, chat, VANI or GitHub.
